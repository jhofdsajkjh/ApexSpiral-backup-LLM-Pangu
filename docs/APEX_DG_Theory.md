# APEX ΔG Theory - 理论白皮书

> 开天辟地，让LLM从混沌走向秩序

## 摘要

APEX ΔG 理论是首个将**香农信息论**与**演化范式**深度融合的LLM防幻觉框架。

通过物理学的三大定律为底座，结合APEX全域演化公式，实现从根源压制幻觉。

---

## 1. 理论底座：香农三大定律

### 1.1 香农信息熵 H(X)

$$H(X) = -\sum_{i} p(x_i) \log_2 p(x_i)$$

**物理含义：**
- 量化LLM输出的不确定性
- 逻辑混乱程度
- 幻觉发散程度

**在盘古中的作用：**
- 直接纳入ΔG第一项
- 熵值越高，ΔG越大，系统越不稳定

### 1.2 香农信道容量 C

$$C = B \cdot \log_2(1 + S/N)$$

**物理含义：**
- 信道最大传输能力
- 记忆承载上限
- 噪声压制能力

**在盘古中的作用：**
- 约束N/C比值
- 当噪声过大时自动触发保护

### 1.3 香农无损编码定理

$$L_{avg} \geq H(X)$$

**物理含义：**
- 平均码长必须不小于熵
- 编码冗余度 r = L_avg / H(X)

**在盘古中的作用：**
- 约束代码/注释/MD规范
- 冗余度异常触发编码修复

## 2. APEX ΔG 全域演化公式

### 2.1 公式定义

$$\Delta G = \lambda \cdot H + \lambda \cdot [(1-F) + e^{-\tau \cdot L} + \delta_B] + \lambda \cdot \frac{N}{C} + \lambda \cdot (1-\Omega) \cdot \nabla E + \lambda \cdot (1-\Gamma) \cdot r + \lambda \cdot (1-\Psi) \cdot \tau + \lambda \cdot (1-\Theta) \cdot H$$

### 2.2 参数说明

| 参数 | 含义 | 范围 | 物理意义 |
|------|------|------|----------|
| H | 信息熵 | [0, +∞) | 输出不确定性 |
| F | 保真度 | [0,1] | 与事实吻合度 |
| L | 逻辑链路 | [0,1] | 推理链完整性 |
| N | 噪声水平 | [0,1] | 干扰程度 |
| C | 信道容量 | (0, +∞) | 记忆承载上限 |
| Ω | 技能覆盖率 | [0,1] | 已覆盖技能占比 |
| ∇E | 进化梯度 | [0,1] | 自我进化速度 |
| Γ | 编码标准率 | [0,1] | 代码规范遵循度 |
| r | 冗余度 | [1, +∞) | 编码效率 |
| Ψ | 情感温度 | [0,1] | 拟人化程度 |
| Θ | 阈值稳定 | [0,1] | 系统稳定性 |
| λ | 归一化系数 | 1.5 | 均衡各模块 |
| τ | 时间衰减 | 0.32 | 信息衰减速率 |
| δB | 基础偏移 | 0.18 | 最小约束 |

### 2.3 七大模块映射

```
ΔG = 模块1(熵) + 模块2(保真+链路+基础)
    + 模块3(信噪) + 模块4(进化)
    + 模块5(编码) + 模块6(情感)
    + 模块7(阈值稳定)
```

## 3. 系统稳态判定

### 3.1 三段式判定

| ΔG范围 | 状态 | 动作 |
|--------|------|------|
| ≤ 2.10 | 健康稳态 | 放行 |
| 2.10~4.0 | 轻度偏离 | 自动局部修复 |
| > 4.0 | 严重越界 | 触发全量自愈 |

### 3.2 健康标准

- 信息熵 H < 0.5
- 保真度 F > 0.8
- 逻辑链路 L > 0.7
- 技能覆盖率 Ω > 0.75
